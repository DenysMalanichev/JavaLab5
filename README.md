# JavaLab5

```Завдання:``` 
Створити клас, який складається з виконавчого методу, що виконує дію над текстовим рядком 
(В усіх питальних реченнях заданого тексту знайти та надрукувати без повторень слова заданої довжини). 
Необхідно обробити всі виключні ситуації, що можуть виникнути під час виконання програмного коду.
Для літер, слів, речень, розділових знаків та тексту створити окремі класи. Слово повинно
складатися з масиву літер, речення з масиву слів та розділових знаків, текст з
масиву речень. Замінити послідовність табуляцій та пробілів одним пробілом.
